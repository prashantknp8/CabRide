# 🚕 CabRide

CabRide is a production-inspired cab booking platform built as a B.Tech Final Year Project.

## Tech Stack

- Java 21
- Spring Boot
- Spring Cloud
- Spring Security
- Spring Data JPA
- MySQL
- Redis
- RabbitMQ
- React
- Docker
- Prometheus
- Grafana

## Architecture

- Microservices
- API Gateway
- Eureka Service Registry
- Database per Service
- Event-Driven Communication

## Services

- Auth Service
- User Service
- Driver Service
- Ride Service
- Payment Service
- Review Service
- Notification Service
